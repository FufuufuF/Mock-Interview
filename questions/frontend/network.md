# 计算机网络

## 基础题

### Q: 请简述 HTTP/1.0、HTTP/1.1 和 HTTP/2 的主要区别
**难度**: 基础
**考察点**: HTTP 协议演进、持久连接、多路复用
**追问方向**:
- HTTP/1.1 的 Keep-Alive 和 HTTP/2 的多路复用有什么本质区别？
- HTTP/1.1 中的队头阻塞（Head-of-Line Blocking）问题是如何产生的？
- HTTP/2 的服务器推送（Server Push）在实际场景中有哪些应用？

### Q: 请描述 TCP 三次握手的过程，并解释为什么需要三次而不是两次
**难度**: 基础
**考察点**: TCP 连接建立、三次握手、可靠传输
**追问方向**:
- 如果第三次握手丢失了会发生什么？
- SYN Flood 攻击的原理是什么？如何防御？
- TCP 四次挥手中为什么需要 TIME_WAIT 状态？TIME_WAIT 过多会有什么问题？

### Q: 请解释浏览器的强缓存和协商缓存的区别，以及各自使用的 HTTP 头部字段
**难度**: 基础
**考察点**: 浏览器缓存策略、Cache-Control、ETag、Last-Modified
**追问方向**:
- Cache-Control 中 no-cache 和 no-store 有什么区别？
- ETag 和 Last-Modified 同时存在时，优先级如何？
- 如何实现一个资源的长期缓存但又能在更新时及时生效（缓存策略设计）？

### Q: 什么是跨域？浏览器的同源策略具体限制了哪些行为？
**难度**: 基础
**考察点**: 同源策略、跨域、CORS
**追问方向**:
- CORS 简单请求和预检请求（Preflight）的区别是什么？触发预检请求的条件有哪些？
- JSONP 的原理是什么？它有哪些局限性？
- 在实际项目中，你用过哪些跨域解决方案？各有什么优缺点？

### Q: Cookie、Session 和 Token 三者的区别与联系是什么？
**难度**: 基础
**考察点**: 身份认证、Cookie、Session、JWT
**追问方向**:
- Cookie 的 HttpOnly、Secure、SameSite 属性分别有什么作用？
- Session 存储在服务端时，分布式环境下如何处理 Session 共享？
- JWT 的组成结构是什么？它有哪些优缺点？

### Q: 请简述 DNS 域名解析的完整过程
**难度**: 基础
**考察点**: DNS 解析、递归查询、迭代查询
**追问方向**:
- 递归查询和迭代查询有什么区别？
- DNS 缓存存在于哪些层级？各自的缓存时间如何控制？
- 前端可以通过什么方式优化 DNS 解析时间（dns-prefetch）？

## 中等题

### Q: 请详细描述 HTTPS 的 TLS 握手过程，以及它是如何保证通信安全的
**难度**: 中等
**考察点**: HTTPS、TLS 握手、对称加密与非对称加密、数字证书
**追问方向**:
- TLS 1.2 和 TLS 1.3 的握手有什么区别？TLS 1.3 做了哪些优化？
- 中间人攻击（MITM）是如何实现的？CA 证书链如何防止中间人攻击？
- 什么是 HSTS？它是如何解决 SSL 剥离攻击的？

### Q: WebSocket 和 HTTP 有什么区别？WebSocket 的连接建立过程是怎样的？

**追问方向**:
- WebSocket 的心跳机制是如何实现的？为什么需要心跳？
- WebSocket 与 SSE（Server-Sent Events）各适用于什么场景？
- 在实际项目中如何处理 WebSocket 的断线重连？

### Q: 请解释 CDN 的工作原理，以及一个用户请求是如何被调度到最近的 CDN 节点的
**难度**: 中等
**考察点**: CDN 原理、DNS 调度、边缘节点、回源策略
**追问方向**:
- CDN 的 DNS 调度和 HTTP 302 调度各有什么优缺点？
- CDN 缓存未命中时的回源流程是怎样的？如何优化回源策略？
- 在前端项目部署中，如何合理利用 CDN 加速？静态资源和 HTML 文件的缓存策略应该有什么区别？

### Q: XSS 攻击有哪些类型？在前端和后端分别如何防御？
**难度**: 中等
**考察点**: XSS 攻击类型、输入过滤、输出编码、CSP
**追问方向**:
- 存储型 XSS、反射型 XSS 和 DOM 型 XSS 的区别是什么？
- Content-Security-Policy（CSP）是如何防御 XSS 的？如何配置？
- React/Vue 框架在 XSS 防御方面做了哪些处理？在什么情况下仍然可能出现 XSS 漏洞？

### Q: CSRF 攻击的原理是什么？请列举几种有效的防御措施
**难度**: 中等
**考察点**: CSRF 原理、SameSite Cookie、CSRF Token、Referer 校验
**追问方向**:
- CSRF Token 的生成和校验流程是怎样的？
- SameSite Cookie 的 Strict、Lax、None 三个值有什么区别？
- CSRF 和 XSS 有什么关系？为什么说 XSS 可以绕过 CSRF 防御？

### Q: TCP 四次挥手的过程是怎样的？为什么断开连接需要四次而建立连接只需要三次？

- TIME_WAIT 状态持续 2MSL 的原因是什么？

### Q: 在大型前端应用中，如何设计一套完善的前端安全防御体系？请从 XSS、CSRF、中间人攻击、供应链安全等多个角度综合阐述

**追问方向**:
