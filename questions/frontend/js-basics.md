## 基础题

### Q1: 请说明 var、let、const 三者的区别，并举例说明各自的使用场景。
**难度**: 基础
**考察点**: 变量声明、块级作用域、变量提升、暂时性死区
**追问方向**:
- var 的变量提升机制具体是怎样的？为什么 let/const 也存在提升但表现不同？
- const 声明的对象属性可以修改吗？如何实现真正的不可变对象？
- 在 for 循环中使用 var 和 let 分别会产生什么结果？请解释原因。

### Q2: JavaScript 中有哪些数据类型？如何准确判断一个变量的类型？
**难度**: 基础
**考察点**: 基本数据类型、引用数据类型、typeof、instanceof、Object.prototype.toString
**追问方向**:
- typeof null 的结果是什么？为什么会这样？
- instanceof 的原理是什么？能否手写一个 instanceof？
- 如何判断一个变量是否为数组？列举尽可能多的方法。

### Q3: == 和 === 有什么区别？请详细描述 == 的隐式类型转换规则。
**难度**: 基础
**考察点**: 隐式类型转换、抽象相等比较算法、ToPrimitive
**追问方向**:
- [] == ![] 的结果是什么？请详细解释转换过程。
- 什么情况下 a == 1 && a == 2 && a == 3 可以为 true？
- Object.is() 和 === 有什么区别？

### Q4: 请解释 JavaScript 中的作用域链是如何工作的。
**难度**: 基础
**考察点**: 全局作用域、函数作用域、块级作用域、词法作用域、作用域链查找
**追问方向**:
- 词法作用域和动态作用域的区别是什么？JavaScript 采用哪种？
- 作用域链的查找过程是怎样的？找不到变量会怎样？
- with 语句和 eval 对作用域有什么影响？为什么不推荐使用？

### Q5: 请解释什么是 Promise，以及它解决了什么问题。
**难度**: 基础
**考察点**: Promise 基本用法、状态机、回调地狱、链式调用
**追问方向**:
- Promise 有哪三种状态？状态转换有什么特点？
- Promise.then() 返回的是什么？如何实现链式调用？
- Promise 构造函数中的代码是同步执行还是异步执行？

### Q6: 箭头函数和普通函数有哪些区别？
**难度**: 基础
**考察点**: 箭头函数、this 绑定、arguments、构造函数
**追问方向**:
- 箭头函数能否作为构造函数使用？为什么？
- 箭头函数中的 this 是在什么时候确定的？
- 箭头函数有 prototype 属性吗？

### Q7: 请说明 ES6 模块化（ESM）和 CommonJS 的区别。
**难度**: 基础
**考察点**: 模块化规范、import/export、require/module.exports、加载机制
**追问方向**:
- ESM 是编译时加载还是运行时加载？CommonJS 呢？这意味着什么？
- ESM 的 import 能否放在 if 语句中？为什么？
- CommonJS 导出的值是拷贝还是引用？ESM 呢？

### Q9: 请解释 JavaScript 的原型链机制，并说明继承是如何实现的。

### Q11: 请解释 JavaScript 的事件循环（Event Loop）机制，包括宏任务和微任务。

### Q12: 请解释 async/await 的原理，它和 Generator 有什么关系？

**考察点**: async/await、Generator、迭代器协议、协程、Promise
**追问方向**:

### Q13: 请解释 ES6 中的 Proxy 和 Reflect，以及它们的应用场景。
