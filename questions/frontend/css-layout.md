# CSS 布局

---

## 基础题

### Q: 请解释 CSS 标准盒模型和 IE 盒模型的区别，以及如何通过 CSS 属性切换两种模型？
**难度**: 基础
**考察点**: 盒模型、box-sizing、width 计算方式
**追问方向**:
- `box-sizing: content-box` 和 `box-sizing: border-box` 分别对应哪种盒模型？
- 在实际项目中，你更倾向于使用哪种盒模型？为什么？
- `margin` 的负值会对盒模型产生什么影响？

### Q: CSS 选择器的优先级是如何计算的？请从高到低列举并说明。
**难度**: 基础
**考察点**: 选择器优先级、权重计算、!important
**追问方向**:
- `!important`、行内样式、ID 选择器、类选择器、标签选择器的权重分别是多少？
- 如果两个选择器权重完全相同，最终生效的是哪一个？
- `:not()` 伪类本身是否参与优先级计算？

### Q: 请列举你知道的 CSS 居中方案，分别适用于什么场景？
**难度**: 基础
**考察点**: 水平居中、垂直居中、flex 居中、绝对定位居中
**追问方向**:
- 对于未知宽高的元素，如何实现水平垂直居中？
- `margin: 0 auto` 居中的前提条件是什么？
- `transform: translate(-50%, -50%)` 配合绝对定位实现居中的原理是什么？

### Q: `position` 属性有哪些取值？请分别说明它们的定位基准和使用场景。
**难度**: 基础
**考察点**: static、relative、absolute、fixed、sticky
**追问方向**:
- `absolute` 定位的元素是相对于哪个祖先元素定位的？
- `sticky` 定位在什么条件下会失效？
- `fixed` 定位的元素一定是相对于视口定位的吗？有没有例外？

### Q: 请简述 CSS3 中 `transition` 和 `animation` 的区别和各自的使用场景。
**难度**: 基础
**考察点**: 过渡、动画、关键帧、触发方式
**追问方向**:
- `transition` 能否实现循环播放的动画效果？
- `animation-fill-mode` 的几个取值分别代表什么含义？
- 如何使用 CSS 动画实现性能最优的平滑动画？应该优先动画哪些属性？

## 中等题

### Q: 什么是 BFC？如何触发 BFC？BFC 能解决哪些实际问题？
**难度**: 中等
**考察点**: BFC 概念、触发条件、margin 塌陷、清除浮动
**追问方向**:
- 请列举至少 4 种触发 BFC 的方式。
- BFC 是如何解决父元素高度塌陷问题的？
- 两个相邻的 BFC 之间的 margin 还会发生折叠吗？

### Q: 请详细介绍 Flex 布局的核心概念，包括容器属性和项目属性。
**难度**: 中等
**考察点**: 主轴/交叉轴、flex-direction、justify-content、align-items、flex 缩写
**追问方向**:
- `flex: 1` 完整写法是什么？`flex: 0 1 auto` 和 `flex: 1 1 0%` 有什么区别？
- 如何使用 Flex 实现一个固定侧边栏 + 自适应内容区的两栏布局？
- `align-self` 和 `align-items` 的关系是什么？

### Q: 请介绍响应式设计的核心原则，以及媒体查询的使用方式和常见断点设计。
**难度**: 中等
**考察点**: 媒体查询、断点、移动优先、视口单位
**追问方向**:
- 移动优先（Mobile First）的设计策略具体是如何体现在 CSS 中的？
- `rem`、`em`、`vw`、`vh` 各自的计算基准是什么？在响应式中如何选择？
- 除了媒体查询，你还知道哪些实现响应式的 CSS 方案？

### Q: 使用 CSS Grid 如何实现一个 12 列栅格系统？请描述关键属性。
**难度**: 中等
**考察点**: grid-template-columns、fr 单位、gap、grid-column
**追问方向**:
- `grid-template-columns: repeat(12, 1fr)` 和 `repeat(auto-fill, minmax(200px, 1fr))` 有什么区别？
- Grid 布局中 `fr` 单位是如何分配剩余空间的？
- Grid 和 Flex 各自更适合哪些布局场景？如何选择？

### Q: 请解释 `z-index` 的层叠上下文是什么？为什么有时设置了较大的 `z-index` 却没有生效？
**难度**: 中等
**考察点**: 层叠上下文、层叠顺序、z-index 生效条件
**追问方向**:
- 哪些 CSS 属性会创建新的层叠上下文？
- 在同一个层叠上下文中，元素的层叠顺序是怎样的？
- `z-index` 为 `auto` 和 `z-index: 0` 有什么区别？

## 困难题

### Q: 请从浏览器渲染原理的角度，解释为什么推荐使用 `transform` 和 `opacity` 做动画，而不是 `width`、`height`、`top`、`left`？
**难度**: 困难
**考察点**: 重排（Reflow）、重绘（Repaint）、合成层（Compositing）、GPU 加速
**追问方向**:
- 浏览器渲染流水线的主要阶段有哪些？
- `will-change` 属性的作用是什么？滥用它会带来什么问题？
- 如何利用 Chrome DevTools 的 Performance 面板和 Layers 面板排查动画性能问题？

### Q: 在大型项目中，如何系统性地解决 CSS 样式冲突和全局污染问题？请对比不同方案的优劣。
**难度**: 困难
**考察点**: CSS Modules、CSS-in-JS、BEM 命名规范、Scoped CSS、CSS 层叠层（@layer）
**追问方向**:
- CSS Modules 的实现原理是什么？它是如何保证类名唯一性的？
- CSS `@layer` 规则是什么？它是如何改变样式优先级管理方式的？
- 在微前端架构下，如何处理不同子应用之间的样式隔离？

### Q: 请实现一个不定数量卡片的瀑布流布局，分别用纯 CSS 和 JS 辅助两种方案，并分析各自的优缺点。
**难度**: 困难
**考察点**: multi-column、Grid 布局、JavaScript 动态计算、性能优化
**追问方向**:
- 纯 CSS 的 `column-count` 方案在排列顺序上有什么局限性？
- 使用 CSS Grid 的 `masonry` 实验性属性的浏览器兼容性如何？
- 在卡片数量很大的情况下，如何结合虚拟滚动来优化瀑布流的渲染性能？

### Q: 请设计一套完整的 CSS 主题切换方案（如深色/浅色模式），要求支持跟随系统、用户手动切换以及持久化存储。
**难度**: 困难
**考察点**: CSS 自定义属性（变量）、prefers-color-scheme、数据属性选择器、localStorage
**追问方向**:
- CSS 自定义属性与预处理器变量（如 Sass 变量）有哪些本质区别？
- 如何避免主题切换时页面出现闪烁（FOUC）？
- 如果需要支持超过两种主题，你的方案如何扩展？
